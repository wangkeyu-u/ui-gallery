# AwardCode Hub · 获奖源码库

自动扫描 `source/` 目录里克隆的获奖源码，按主题分类，提供**真实代码预览**与**源码浏览器**。所有展示/服务的代码都来自 `source/` 中的原始文件，**与获奖代码逐字节一致**（源码浏览器/预览不做任何内容改写，仅预览传输时对 obys 的绝对路径做 `/p/:id/` 前缀重写以便 iframe 正确加载）。

## 运行

```bash
cd hub
node server.js        # 默认端口 8200，可用 PORT=xxxx 覆盖
# 浏览器打开 http://127.0.0.1:8200
```

零依赖，只需 Node.js（无需 npm install）。

## 功能

- **主题筛选**：3D/WebGL、平滑滚动、作品集、电商、创意机构、极简、图像特效、产品落地页。
- **自动收录**：把任意仓库浅克隆进 `../source/`，刷新页面即自动出现（无需改代码）。
- **预览真实代码**：iframe 直接加载 `source/` 里的原始文件运行（静态站即时可跑；已构建的模板直接预览；未构建的 React/Next 项目给出本地运行命令）。
- **查看源码**：树形浏览 + 逐文件查看/复制，内容即为磁盘上的原文件。
- **复制运行命令**：一键复制各项目的启动命令。

## 接口

- `GET /api/projects` — 项目列表（含主题、年份、类型、体积）
- `GET /api/projects/:id/tree?path=` — 目录树
- `GET /api/projects/:id/raw?path=` — 原始文件内容（二进制返回 415）
- `GET /p/:id/...` — 静态预览真实代码
